Basic Web UI to interact with Solidity based Ethereum blockchain.


## Getting Started

Open `pages/index.js` and set your contract Address inside `ADDRESS` variable.

Rename `WavePortal.json.sample` to `WavePortal.json` inside `utils` folder and copy the content from your original WavePortal.json file in the Solidity App.

install the dependencies

```bash
npm install
```

Finally run the development server:

```bash
npm run dev
# or
yarn dev


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
